# testing-practice-jest

Write some tests using the JavaScript test runner Jest.


## Built With

- Javascript
- Webpack
- Jest

## How to Run the App Locally
```
   git clone https://github.com/RoodzFernando/testing-practice-jest
   cd testing-practice-jest
   run npm test
```
